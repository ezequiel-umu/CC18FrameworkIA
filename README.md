## Pasos para subir la primera versión del bot

1. Clonar el repositorio
```
git clone https://github.com/ezequiel-umu/CC18FrameworkIA
```
2. Crear una copia en .tar.gz
```
cd CC18FrameworkIA/bot
make zip
```
3. Subir `bot.tar.gz` a la web.
